HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6>

Defines HTML headings. <h1> is the highest level, <h6> is the lowest.

<h1>Heading 1</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://example.com">This is a link</a>

<img>

Defines an image.

<img src="image.jpg" alt="Description">

<ul> and <li>

Defines an unordered list and list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> and <li>

Defines an ordered list and list items.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div>

Defines a division or section in an HTML document.

<div>Content here</div>

<span>

Defines a section in a document, used for styling.

<span style="color:red">Red text</span>

<form>

Defines an HTML form for user input.

<form>
    <input type="text" placeholder="Enter text">
    <input type="submit">
</form>

<table>

Defines a table.

<table>
    <tr>
        <th>Header</th>
        <th>Header</th>
    </tr>
    <tr>
        <td>Data</td>
        <td>Data</td>
    </tr>
</table>

CSS Properties

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

div {
    background-color: lightgray;
}

font-size

Sets the size of the font.

h1 {
    font-size: 24px;
}

margin

Sets the margin area around elements.

p {
    margin: 10px;
}

padding

Sets the padding area inside an element.

div {
    padding: 15px;
}

border

Sets the border around an element.

table {
    border: 1px solid black;
}

display

Specifies the display behavior of an element.

span {
    display: inline-block;
}

text-align

Sets the horizontal alignment of text.

h2 {
    text-align: center;
}
