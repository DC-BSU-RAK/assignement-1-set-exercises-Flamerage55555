HTML & CSS Quick Reference Guide

HTML Tags

Headings

Defines headings from <h1> to <h6>.

<h1>Heading 1</h1>

Paragraph

Defines a paragraph.

<p>This is a paragraph.</p>

Link

Defines a hyperlink.

<a href="https://example.com">Visit</a>

Image

Defines an image.

<img src="image.jpg" alt="Description">

CSS Properties

Color

Sets the text color.

p { color: blue; }

Background

Sets the background color.

div { background-color: lightgray; }

Font Size

Sets the font size.

h1 { font-size: 24px; }
